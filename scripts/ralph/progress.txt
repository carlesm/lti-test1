## Codebase Patterns
- Use `uv run python manage.py ...` instead of `python manage.py ...` (uv-managed environment)
- Django project name: `lti_project_selection`; main app: `projects`
- Settings reads SECRET_KEY, DATABASE_URL, ALLOWED_HOSTS from env; SQLite fallback for dev
